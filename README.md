# RISC-V RV32I RTL Design & Verification

A professional RTL Design and Verification project implementing a 32-bit RV32I RISC-V processor using Verilog/SystemVerilog.

## Project Goals

- RTL Design
- 5-Stage Pipeline
- Hazard Detection
- Forwarding Unit
- SystemVerilog Verification
- Assertions (SVA)
- Functional Coverage
- Synopsys VCS/Verdi Simulation
